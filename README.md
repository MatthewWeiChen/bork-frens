# bork-frens
A full stack application for dog owners who want a community to find doggy play pals for their fur babies.

## Technologies Used
* React
* Node.js
* JavaScript ES6
* HTML5 & CSS3
* Express
* PostgreSQL
* Bootstrap 4
* Pgweb
* AWS EC2

## Live Demo
https://borkfrens.mattweichen.tech/

## Features
:dog: -Features I personally worked on.

- User can see dropdown menu of users to login
- User can navigate to homepage
- User can edit own profile page
- User can view own profile page
- User can view list of conversations  :dog:
- User can view conversation with other user :dog:
- User can direct message with other user :dog:
- User can view list of connection requests
- User can accept or deny connection request
- User can view other user's profile
- User can request connection to other user
- User can see map (static) and number of nearby users
- User can see list of nearby users
- User can navigate with side navigation menu
- User can navigate with navigation back button

### Stretch Features
- User can update account settings
- User can sign up for account
- User can upload own profile picture
- User can login to account with username and password

## Getting Started
1. Clone the repository.
```
git clone https://github.com/MatthewWeiChen/bork-frens.git
```
2. Install all dependecies with NPM.
```
npm install
```
3. Start the project. Once started you can view the application by opening http://localhost:3000 in your browser.
```
npm run dev
```
